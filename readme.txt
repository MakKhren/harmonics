Harmonics
Автор MakKhren
Програма для расчета коэфициентов искажения тока и напряжения,  для учета влияние преобразовательной техники на электрические сети на этапе проектирования.
Для расчета принимаем что преобразователи работают по схеме Ларионова, функцию тока прямоугольной формы, нечетная

метод расчета принят по на основании
В.С. Осипов "МЕТОД РАСЧЕТА КОЭФФИЦИЕНТА ИСКАЖЕНИЯ СИНУСОИДАЛЬНОСТИ КРИВОЙ НАПРЯЖЕНИЯ,
СОЗДАВАЕМОГО ТРЕХФАЗНЫМИ ВЫПРЯМИТЕЛЯМИ."
